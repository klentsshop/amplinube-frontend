# 🚨 Plan de Contingencia - Protocolos de Emergencia
*Este documento indica qué hacer ante fallas técnicas para no detener la operación del restaurante.*

---

## 📵 Escenario A: Falla de Internet
* **Diagnóstico:** El sistema no guarda las órdenes o se queda cargando indefinidamente.
* **Acción:** 1. **No cierre la pestaña del navegador.**
    2. Active el "Anclaje de red" o "Hotspot" desde un teléfono celular con datos.
    3. Conecte la tablet o computador a la red del celular.
    4. El sistema reanudará la conexión automáticamente y podrá guardar los pedidos pendientes.

## 🖨️ Escenario B: Falla de Impresora (Cocina/Ticket)
* **Diagnóstico:** El pedido se guarda en el sistema pero no sale el papel.
* **Acción:**
    1. Verifique que la impresora tenga papel y no tenga luces rojas de error.
    2. Revise la conexión de red/USB de la impresora.
    3. **IMPORTANTE:** Los pedidos **SÍ están guardados** en el sistema. Puede dictar el pedido desde la pantalla o presionar "Reimprimir" una vez solucionado el problema físico.

## ⚡ Escenario C: Corte de Energía
* **Acción:** 1. Si usa tablets o laptops, el sistema seguirá funcionando por la batería de los equipos. 
    2. Use datos móviles para mantener la sincronización.
    3. Para la impresión, deberá esperar a que retorne la energía o entregar facturas manuales temporalmente anotando el número de orden del sistema.

## 🔑 Escenario D: Olvido de PIN
* **Acción:** Contacte de inmediato al soporte técnico de Kelly Johanna Rodriguez para el restablecimiento manual desde la base de datos central.

**Nota:** El sistema está diseñado para proteger los datos. Aunque el hardware falle, su historial de ventas permanece seguro en la nube.
