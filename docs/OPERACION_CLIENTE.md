# 📖 Manual de Operación - Pedidos Pro
**Versión del Usuario:** 1.0  
**Desarrollado por:** Kelly Johanna Rodriguez 

---

## 1. Inicio de Jornada
* **Acceso:** Abra el navegador e ingrese a la dirección del sistema.
* **Identificación:** Seleccione su nombre de la lista de meseros activos.
* **Caja:** Si es el encargado de caja, asegúrese de tener acceso con su PIN correspondiente.

## 2. Gestión de Pedidos (Mesas)
* **Abrir Mesa:** Toque el número de mesa, seleccione los productos y presione **GUARDAR**. El pedido se enviará automáticamente a la impresora de cocina.
* **Agregar Productos:** Seleccione una mesa ocupada (color resaltado), añada los nuevos ítems y presione **ACTUALIZAR**.
* **Notas Especiales:** Use el icono de comentario en cada producto para especificar cambios (ej: "Sin cebolla", "Bien asado").

## 3. Proceso de Pago
* **Pre-cuenta:** Antes de cobrar, presione **IMPRIMIR TICKET** para entregar el detalle al cliente.
* **Registro de Pago:** Una vez recibido el dinero, elija el método (**Efectivo, Digital o Tarjeta**) y presione **COBRAR**. 
* **Liberación:** La mesa se pondrá en color blanco (libre) automáticamente tras confirmar el cobro.

## 4. Gastos y Cierre
* **Registrar Gasto:** Use el botón de "Gastos" para anotar salidas de dinero (ej: compras urgentes, proveedores).
* **Cierre de Caja:** Al finalizar el día, ingrese al módulo de Reportes para ver el resumen total de ventas y el saldo final.
