## 📋 Descripción General
Sistema de Punto de Venta (POS) profesional diseñado para optimizar el flujo de pedidos, cocina y facturación en tiempo real. Construido con una arquitectura moderna y escalable.

## 🛠️ Documentación de Soporte
Para una gestión eficiente, consulte los manuales especializados en la carpeta `/docs`:

1.  **[Guía de Instalación](./docs/INSTALACION.md):** Pasos técnicos para configurar nuevos clientes.
2.  **[Operación del Cliente](./docs/OPERACION_CLIENTE.md):** Manual de uso para meseros y administradores.
3.  **[Plan de Contingencia](./docs/PLAN_CONTINGENCIA.md):** Protocolos ante fallos de internet o hardware.
4.  **[Respaldo y Legal](./docs/RESPALDO_Y_LEGAL.md):** Términos de responsabilidad y manejo de datos.

## ⚙️ Tecnologías Principales
* **Frontend:** Next.js 14+ (App Router)
* **Estado Global:** React Context API
* **Base de Datos:** Sanity.io (Headless CMS)
* **Estilos:** CSS Modules & Responsive Design

## 🔒 Seguridad y Versión
Este software cuenta con una capa de blindaje de datos mediante adaptadores (`lib/dataAdapter.js`) y gestión de roles por PIN (Admin/Cajero).
